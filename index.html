The header header does not have any positioning applied, therefore assumes the default - which is static

The nav bar position is fixed, which means it stays towards the top of the page even if you scroll down. It stays in the same position.

The articles position is relative which means it keeps the normal layout flow. This will allow the element to move but still consume space in the orginal position.

This article position is also relative but with more displacements that way it appears below the other article.

The aside position is absolute, so it takes the element out of normal document flow and positions at top/right/left/bottom placements.

The footer position is fixed to 0 px that way it stays at the bottom of the page no matter where you scroll.
